## 크루관리

1. 크루 추가 가능 (동일한 이름X, 이름 최대 5글자)
2. 코스 별로 크루 삭제 가능 (Confirm을 사용)
3. 크루 table의 첫번째 열에는 index를 넣어 순서를 표시

## 팀 매칭 관리 기능

1. 코스별로, 미션마다, 무작위로 팀 매칭
2. 1팀당 최소 인원 수 입력
3. 쉼표로 구분

## 기타 요구사항

1. index.html파일은 수정할 수 없다.
2. 예외 발생 상황은 alert메서드를
